# Calendar Weeks

A minimal, single-page calendar showing week numbers for any year. Click or drag to highlight weeks. No dependencies, printable.

**[Live demo](https://sa.ndropad.in/calendar-weeks/)**

## Features

- Full-year calendar with week numbers
- Click a date to highlight its week row
- Click and drag to highlight a range of weeks
- Navigate between years
- Print-friendly layout
- Works entirely in a single HTML file â€” no build step, no dependencies

## Development

Just open `index.html` in a browser. That's it.

## Deployment

Hosted with GitHub Pages. Any push to `main` deploys automatically.
